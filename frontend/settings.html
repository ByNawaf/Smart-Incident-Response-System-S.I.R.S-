Quick Start

1. Start Backend
Run start.bat or py -m uvicorn main:app --reload in the backend folder
2. Open Browser
Navigate to http://localhost:8000
3. Run Demo
Click the "Run Live Demo" button on the Dashboard or Judges View
4. Start Ollama
Run ollama serve, then ollama pull qwen3:8b
